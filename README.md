Cloud-native DevSecOps project implementing a CI/CD pipeline with Jenkins, Docker, Kubernetes, and SIEM-style log monitoring (ELK stack).

---

## 🚀 Tech Stack
- AWS EC2 (IaaS)
- Jenkins (CI/CD)
- Docker (Containerisation)
- Kubernetes (Orchestration)
- ELK Stack (Monitoring & Logging)

---

## 📂 Project Structure
- app/ → Application & Dockerfile
- k8s/ → Kubernetes manifests
- jenkins/ → CI/CD pipeline
- monitoring/ → ELK stack configs
- docs/ → Documentation
- diagrams/ → Architecture diagrams
